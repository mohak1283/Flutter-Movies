## Getting Started

For help getting started with Flutter, view online
[documentation](https://flutter.dev/).

#### 1. [Setup Flutter](https://flutter.dev/setup/)

#### 2. Clone the repo

```sh
$ git clone https://github.com/mohak1283/Flutter-Movies.git
$ cd Fluttery-Movies/
```

#### 4. Run the app

```sh
$ flutter run
```
